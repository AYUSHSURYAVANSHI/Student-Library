HTML

HTML stands for HyperText Markup Language.HTML is used to create the structure of web pages that are displayed on the World Wide Web (www). When it comes to HTML, it's written structure has two components- tags and attributes.

Basic HTML tags

I'll be listing out the basic types of tags you would require to get started with HTML. But before that, where can you write HTML?
You can write HTML in any text editor like notepad so long as you add the .html extension so that the computer can understand that its written in html. You can also use other code editors like VSCode, Sublime Text etc.,

An HTML element is defined by a start tag, some content, and an end tag.
